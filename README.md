- 음식점 List API : [GET] restaurant/list?page=int
- 음식점 Detail API : [GET] restaurant/detail?restaurant_id=int
- 음식점 Create API : [POST] restaurant
- 메뉴 Create API : [POST] restaurant/menu?restaurant_id=int
